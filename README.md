## Описание проекта
Цель игры набрать как можно больше очков убивая противников. 

Противнике в игре делятся на 3 типа:
  - Ближняя дистанция(Демон)
  - Средняя дистанция(Солдат)
  - Дальняя дистанция(Солдат)
